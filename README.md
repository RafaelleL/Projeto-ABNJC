# Site Institucional – Associação Beneficente Natal Justino da Costa
Este projeto consiste no desenvolvimento de um site institucional para a Associação Beneficente Natal Justino da Costa, com o objetivo de divulgar suas ações sociais, facilitar a captação de recursos e permitir o cadastro de voluntários.

## Contexto
O site foi desenvolvido como parte de um projeto de extensão universitária, buscando contribuir com a inclusão digital e o fortalecimento das ações sociais da associação junto à comunidade local.

## Funcionalidades
- Página inicial com apresentação da ONG
- Página "Sobre Nós"
- Divulgação de ações e projetos sociais
- Informações para doações
- Informações de contato
- Cadastro de voluntários (em desenvolvimento)

## Tecnologias Utilizadas
- HTML5
- CSS3
- Git e GitHub
